## Overview
This project provides a FastAPI-based REST API for speech classification.
It detects whether an input speech audio file is generated by AI or by a human.

## Installation
<b>1. Create and activate a virtual environment</b>

Linux / macOS:
``` bash
python3 -m venv venv
source venv/bin/activate
```

Windows:
``` bash
python -m venv venv
venv\Scripts\activate
```

<b>2. Install Python dependencies</b>
``` bash
pip install -r requirements.txt
```

## Running the API

Start the FastAPI server using Uvicorn:
``` bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

The API will be available at:
- http://localhost:8000

Interactive API documentation:
- http://localhost:8000/docs
